# easyjump.tmux

EasyMotion for Tmux

# Requirements

- Python >= 3.5
- Bash >= 5.0
- GNU Coreutils

# Usage

Set up the key binding in Tmux (Default: `ctrl+b j`):

```sh
./easyjump.tmux
```
